# Travel Pin Map (0원/개인용/로그인 없음)

지도 메인 + 좌측 폴더(도시 > 테마) + 핀 저장/검색/내 위치/길찾기(구글맵).

## 로컬 실행
```bash
npm install
npm run dev
```

## GitHub + Vercel 배포
1) GitHub에 새 repo 만들고 이 프로젝트 전체를 push
2) Vercel → New Project → GitHub repo 선택 → Deploy

## 주요 기능
- 한국/일본 영역 밖으로 이동 제한
- 장소/주소/상호 검색: Nominatim(OSM) (개인용 수준)
- 검색 결과 클릭 → 지도 이동 + 바로 저장
- 핀 저장: 이름(기본값 자동/수정 가능), 주소(한국어/일본어 자동), 메모, 링크 여러 개, 사진 업로드(캡쳐 등) + 즉시 미리보기
- 도시/테마: 추가/이름 수정/삭제 가능, 원하는 대로 열고 닫기(도시 여러 개 동시에 펼침 허용)
- 길찾기: 구글맵 버튼(대중교통/도보/차량)
- 도보/차량 ETA: OSRM 공개 서버(개인용 수준)
- 데이터 저장: 브라우저 IndexedDB (Dexie)

## 주의
- “실시간 운영정보(영업중/휴무)”는 무료/무키 방식으로 안정 제공이 어렵습니다.
  기본은 [구글맵 열기] 버튼으로 확인하도록 구성했습니다.
- 사진 업로드는 브라우저 로컬 저장(IndexedDB)이라, 너무 큰/너무 많은 이미지는 저장 용량에 영향을 줄 수 있습니다.
